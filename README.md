# 🧬 HSP70 Multiple Sequence Alignment Using Clustal Omega

## 📌 Overview

This project demonstrates Multiple Sequence Alignment (MSA) of five Heat Shock Protein 70 (HSP70) sequences from different organisms using Clustal Omega. The alignment was performed to identify conserved regions, conserved motifs, and sequence similarities among HSP70 proteins.

---

## 🎯 Objectives

- Retrieve HSP70 protein sequences from UniProt.
- Perform Multiple Sequence Alignment using Clustal Omega.
- Identify conserved regions among the aligned sequences.
- Identify conserved motifs in the HSP70 protein family.
- Interpret the biological significance of the alignment.

## 🔄 Workflow

UniProt → Download FASTA Sequences → Clustal Omega → Multiple Sequence Alignment → Conserved Region & Motif Analysis

---

## 📊 Key Findings

- Successfully aligned five HSP70 protein sequences.
- Identified highly conserved amino acid regions.
- Observed the conserved HSP70 signature motif (IDLGTTYS).
- The alignment indicates strong evolutionary conservation among HSP70 proteins.
